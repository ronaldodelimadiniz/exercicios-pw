📌 Lista de Exercícios – Estrutura Básica do HTML
Instruções:

Crie um arquivo HTML para cada exercício (ex.: exercicio1.html) e salve todos dentro da pasta lista-1.
Utilize a estrutura básica do HTML.
Complete os exercícios abaixo dentro dos respectivos arquivo.
📝 Exercício 1 – Página de Apresentação
Crie uma página que contenha:

Um título principal com seu nome.<h1>
Um parágrafo falando um pouco sobre você.<p>
Uma lista não ordenada com três coisas que você gosta de fazer.<ul>
📌 Exercício 2 – Lista de Compras
Crie uma lista ordenada com pelo menos 5 itens que você compraria se ganhasse um vale-compras ilimitado.<ol>

📌 Exercício 3 – Meus Filmes ou Séries Favoritos
Crie uma página que contenha:

Um título falando sobre seus filmes ou séries favoritas.<h1>
Um parágrafo explicando por que você gosta dessas obras.<p>
Uma lista com pelo menos 3 filmes ou séries.<ul>
📌 Exercício 4 – Criando um Pequeno Blog
Crie uma página HTML que contenha:

Um título principal com o nome do blog (invente um nome).<h1>
Três postagens, cada uma contendo:
Um título com o nome do post.<h2>
Um parágrafo com um texto curto sobre o tema.<p>
Uma lista ou com informações relacionadas ao post.<ul><ol>
📌 Desafio Extra – Página Completa
Crie uma página completa com:

Um título .<h1>

Um parágrafo explicando o conteúdo da página.<p>
Uma lista ou .<ul><ol>
